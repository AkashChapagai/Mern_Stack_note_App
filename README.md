# MERN Notes App

A full-stack notes application built with the MERN stack. Users can create, read, update, delete, and mark notes as done. The app includes a polished animated UI, MongoDB Atlas database integration, loading/error handling, and a clean frontend structure.

## Tech Stack

- React
- Vite
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- CSS animations

## Features

- Create notes
- View saved notes
- Edit notes
- Delete notes
- Mark notes as done
- Animated completed-note effect
- Loading and error handling
- Responsive modern UI
- Backend API connected to MongoDB Atlas
